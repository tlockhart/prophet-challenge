# Mercado Libre Forecasting Net Prophet

## Summary

This repository contains a Jupyter Notebook that demonstrates the process of forecasting search trends for Mercado Libre. With over 200 million users, Mercado Libre is the most popular e-commerce site in Latin America. The purpose of this project, is to use the Model Profit Predict Workflow to forecast Mercado Libreâ€™s search traffic, using the Prophet opensource model for time series forecasting.  The project will leverage this data to predict patterns and potentially forecast stock prices for Mercado Libre.

The instructions for this Challenge are divided into four steps, as follows:

1. Find unusual patterns in hourly Google search traffic.
2. Mine the search traffic data for seasonality.
3. Relate the search traffic to stock price patterns.
4. Create a time series model with Prophet.
---

## Setup

The Jupyter Notebook for this project is named `forecasting_net_prophet.ipynb`. Ensure you have the required dependencies installed before proceeding.

### Requirements

1. Python v3.10 or later
2. Jupyter Notebook
3. Visual Studio Code or any other preferred IDE
4. Google Gmail account
